# GUI-008 Application Guide

## Unified Experience

With GUI-008, oMLX merges previously independent workspaces into a single desktop application featuring persistent window state, global search (⌘K), comprehensive accessibility support, and strict structural boundaries.

## Key Features
- **Persistent State:** Uses `WindowStateManager` to retain selected sessions, workspaces, and window sizes.
- **Global Search:** Command Palette available anywhere via ⌘K to quickly navigate or search models, sessions, and diagnostics.
- **Preferences Scene:** First-class macOS preferences pane (⌘,) managing appearance, accessibility, and editor defaults.
- **Strict Layers:** Enforces MVVM and Service protocols, eliminating View-level networking.
